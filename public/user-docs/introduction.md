# Welcome to the Social Automation Platform

Our platform provides enterprise-grade infrastructure to automate fleets of iOS and Android devices across multiple social media applications safely, concurrently, and reliably. 

## Key Concepts

- **Devices**: Physical smartphones or emulators connected to our orchestration worker via USB (Android) or iproxy (iOS).
- **Accounts**: Individual social media profiles assigned to daily execution budgets and tracking their "warm-up" reputation states to avoid detection.
- **Macros**: Reusable JSON structures that define step-by-step UI automation routines (likes, swipes, wait delays).
- **Runs**: An execution instance of a Macro targeting a specific Account on a specific Device.

## Navigating the Dashboard

- **Social Dashboard**: Your high-level metrics, active account statuses, and automated warm-up progression tracks.
- **Fleet Health**: Real-time status of your connected physical devices and connection bridges.
- **Macros**: The authoring environment where you build and update automation sequences.
